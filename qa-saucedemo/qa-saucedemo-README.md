## Project Overview

This repository contains a complete manual testing project for the SauceDemo web application.

The purpose of this project is to demonstrate a structured software testing process, from test planning and requirements analysis to test execution, defect reporting, and final test summary. All documentation has been created following common QA practices used in professional software development environments.

---

## Application Under Test

**Application:** SauceDemo

**URL:** https://www.saucedemo.com/

**Application Type:** E-commerce Web Application

**Test Type:** Manual Functional Testing

---

## Project Objectives

The main objectives of this project are to:

- Verify core application functionality
- Validate user workflows
- Identify functional defects
- Produce clear QA documentation
- Demonstrate professional manual testing practices
