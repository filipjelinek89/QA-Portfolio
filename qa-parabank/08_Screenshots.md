This document provides an index of screenshots captured during manual testing of the ParaBank application.

Screenshots are used as supporting evidence for:

* Test execution
* Bug reports
* UI validation
* Functional verification

# 4. Naming Convention

Use the following naming convention for screenshot files:

```text
SS001_Login_Page.png
SS002_Successful_Registration.png
SS003_Transfer_Funds.png
SS004_BUG001_Invalid_Transfer.png
```

Benefits:

* Easy to locate evidence.
* Direct traceability to test cases and bug reports.
* Consistent organization throughout the project.

# Notes

* Capture screenshots only when they provide useful evidence.
* For failed test cases, include screenshots whenever possible.
* Ensure screenshots do not contain sensitive personal information.
* Use clear filenames that correspond to the related test case or bug report.
