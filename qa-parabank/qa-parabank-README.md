# 🏦 ParaBank Manual Testing Project

A comprehensive manual testing project focused on a web-based online banking application.

The objective of this project is to demonstrate professional QA documentation and testing techniques while validating critical banking features such as user registration, authentication, account management, bill payments, and money transfers.

This project simulates the work of a Junior QA Engineer testing a real banking system.

# 🎯 Project Objectives

The purpose of this project is to verify that the banking application performs reliably and securely under normal user conditions.

Testing focuses on validating:

- User Registration
- User Login
- Account Overview
- Account Activity
- Open New Account
- Transfer Funds
- Bill Payment
- Request Loan
- Account Logout
